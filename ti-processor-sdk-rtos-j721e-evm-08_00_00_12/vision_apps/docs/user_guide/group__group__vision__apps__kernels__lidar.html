Creates a Lidar Ground Plane Classification (GPC) Node.

The node receives a 3d point cloud, estimates normal vectors and determines which points are believed to be on the ground plane based on their normal vector. Points are tagged as "on the ground" or "not on the ground" in the output 3d point cloud.

In addition, the output point cloud is transformed to the current INS position with the origin provided by root_ecef.

Therefore, PTK_INS module needs to be enabled in the system.

Parameters
[in]graphThe reference to the graph.
[in]configurationKernel configuration. Array element type tivx_lidar_gpc_config_t
[in]point_cloud_inArray element type PTK_PointCloud
[in]lidar_metaArray element type PTK_LidarMeta
[in]m_ego_lidarArray element type PTK_RigidTransform
[in]root_ecefArray element type PTK_Position
[out]point_cloud_outArray element type PTK_PointCloud
[out]normal_cloudArray element type PTK_PointCloud
Return values
vx_nodeA node reference. Any possible errors preventing a successful creation should be checked using vxGetStatus

Creates a Lidar Motion Distortion Correction (MDC) Node.

The node receives a raw lidar 3d point cloud and uses INS motion measurements together with the lidar-to-INS(ego) calibration to correct the 3d point cloud for motion distortion caused by the rotation of the Velodyne lidar.

PTK_INS module needs to be enabled in the system.

Parameters
[in]graphThe reference to the graph.
[in]point_cloudArray element type PTK_PointCloud
[in]metaArray element type PTK_LidarMeta
[in]m_ego_lidarArray element type PTK_RigidTransform
[in]root_ecefArray element type PTK_Position
[out]mdc_point_cloudArray element type PTK_PointCloud
Return values
vx_nodeA node reference. Any possible errors preventing a successful creation should be checked using vxGetStatus
